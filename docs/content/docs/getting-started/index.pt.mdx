## O que é o Pix.js?

Pix.js é um conjunto de ferramentas para trabalhar com PIX em seus projetos JavaScript/TypeScript.
O projeto se dividido nos seguintes pacotes:

<Cards>

<Card icon={<CpuIcon className="text-purple-300" />} title='@pix.js/core'>
Lógica principal e componentes para validações PIX e geração de QR code.
</Card>

<Card icon={<QrCodeIcon className="text-purple-300" />} title='@pix.js/qrcode' href='/docs/getting-started/qrcode'>
Fábrica e analisador de QR codes para QR codes PIX estáticos e dinâmicos.
</Card>

<Card icon={<CheckCheckIcon className="text-purple-300" />} title='@pix.js/validator' href='/docs/getting-started/validators'>
Validador para chaves PIX, checksums CRC e outros dados relacionados ao PIX.
</Card>

<Card icon={<BookAIcon className="text-purple-300" />} title='@pix.js/dict' href='/docs/getting-started/dict'>
SDK da API DICT para trabalhar com a API do DICT (Diretório de Identificadores de Contas Transacionais) do Brasil.
</Card>

</Cards>
