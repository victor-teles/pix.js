## What is Pix.js?

Pix.js is a set of tools for integrating PIX payments into your JavaScript/TypeScript projects.
It has different packages:

<Cards>

<Card icon={<CpuIcon className="text-purple-300" />} title='@pix.js/core'>
Core logic and components for PIX validations and QR code generation.
</Card>

<Card icon={<QrCodeIcon className="text-purple-300" />} title='@pix.js/qrcode' href='/docs/getting-started/qrcode'>
QR code factory and parser for both static and dynamic PIX QR codes.
</Card>

<Card icon={<CheckCheckIcon className="text-purple-300" />} title='@pix.js/validator' href='/docs/getting-started/validators'>
Validator for PIX keys, CRC checksums, and other PIX-related data.
</Card>
