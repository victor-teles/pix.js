## Validate a PIX key

To validate a PIX key, you can use the `validatePixKey` function.

```ts
import { validatePixKey } from "@pix.js/validator";

const isValid = validatePixKey("1234567890");
```

To install the Validator package, you can use your favorite package manager:

```bash
npm install @pix.js/validator
```
