## Installation

To install the QRCode package, you can use your favorite package manager:
<Tabs groupId='package-manager' persist items={['npm', 'pnpm', 'yarn', 'bun']}>

```bash tab="npm"
npm install @pix.js/qrcode
```

```bash tab="pnpm"
pnpm add @pix.js/qrcode
```

```bash tab="yarn"
yarn add @pix.js/qrcode
```

```bash tab="bun"
bun add @pix.js/qrcode
```

</Tabs>
