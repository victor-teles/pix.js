<div align="center"><strong>Pix.js is a set of tools for interacting with Pix and Dict API.</strong></div>

<br />

## Installation

Working with qrcode:

```bash
npm install @pix.js/qrcode
```

Working with validators:

```bash
npm install @pix.js/validator
```

Working with dict:

```bash
npm install @pix.js/dict
```
